# 🏃‍♂️ CyberJogger Endless Runner HTML Game

An App I create for nieces and nephews.

Cyberjogger is a high-intensity, 3D synthwave endless runner built entirely in a single HTML file. Dodge neon obstacles, collect energy cores, and survive the accelerating cyber-grid!

As you progress, the game tracks simulated fitness metrics (Calories and Heart Rate) and ramps up the difficulty with dynamic zones and moving targets. Can you handle the Overdrive zone?

## ✨ Features

⚡ Fluid 3D Movement: Smooth, physics-based lateral gliding with dynamic camera tilting for an immersive sense of speed.

🌆 Synthwave Aesthetics: Infinite scrolling neon grids, distance fog, and glowing primitives powered by Three.js.

🧬 Dynamic Difficulty (Zones): * Zone 1 (0m - 500m): Standard dodging and weaving.

Zone 2 (500m+): The grid turns red, speeds increase, and obstacles begin to slide unpredictably across lanes.

Zone 3 (1500m+): Overdrive mode. Blinding speeds and high-frequency hazards.

🛡️ Power-Ups: Collect green Energy Cores to stay alive, or grab the rare Gold Immunity Core to gain a 5-second shield that lets you smash through obstacles for bonus points!

🎵 Procedural Audio: Fully self-contained Web Audio API synthesizer. Features a pulsing background drone, collection chimes, and retro crash sounds without needing any external audio files.

🏆 High Score Tracking: Keeps track of your best distance and calories burned during your session.

## 🎮 Controls

The game features fully responsive controls for both Desktop and Mobile devices.

Desktop: * Use the Left / Right Arrow Keys or A / D to glide smoothly between lanes.

Mobile / Touch: * Tap and Drag left or right anywhere on the screen for 1:1 fluid lane control.

## 🛠️ Tech Stack

HTML5 / CSS3 (Responsive UI overlays)

Vanilla JavaScript (Game logic, procedural generation, Web Audio API)

Three.js (3D Rendering, lighting, and particle effects)

## 🚀 How to Run

Because the game is built completely in a single file with no external visual or audio assets, installation is instantaneous:

Clone or download this repository or visit: https://jimliddle.github.io/Neon-Fit-Runner-Game/

Open index.html in any modern web browser.

Click Start Workout and enjoy!
